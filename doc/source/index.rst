吊舱文档
==============================


智能吊舱=云台+相机+AI芯片+人机交互软件+深度学习

我们建议第一次使用吊舱的朋友，通过文档左边的目录找到对应产品并逐章节阅读。无人车使用及开发需要用户有一定技术基础，需要使用者掌握一定的Linux/ROS/C++/Python/PX4基本知识。

祝您使用愉快！
